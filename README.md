# 📊 ETL Pipeline: End-to-end Job Market Trends Pipeline
## 📌 Project Overview

This project implements an End-to-End ETL (Extract, Transform, Load) Pipeline using:

Web Scraping (data extraction)

Data Staging

Data Processing with Apache Spark

Loading into a Data Warehouse

The pipeline is divided into two main parts:

Part 1 — Scraping to Staging

Part 2 — Processing to Warehouse
